# Funcionalidades

E suas respectivas regras

Legenda:
    - <span style="color: rgb(200,0,0);">AD</span>: A definir

Nota 1: Arquivos de áudio e gravações podem ser substituídos por "som" a qualquer momento, para poupar palavras

### <span style="color: rgb(218,112,214);">Adicionar arquivo de áudio</span>

<span style="color: rgb(218,112,214);">**Descrição**</span>: O usuário poderá enviar um arquivo de áudio para o aplicativo
Formatos prioritários:
- WAV
- MP3
Formatos previstos:
- OGG/OGA
- FLAC
- M4A
<span style="color: rgb(218,112,214);">**Regras**</span>:
    - Tamanho de arquivo (<span style="color: rgb(200,0,0);">AD</span>)
    - Duração de arquivo (<span style="color: rgb(200,0,0);">AD</span>)



### <span style="color: rgb(218,112,214);">Gravar áudio</span>

<span style="color: rgb(218,112,214);">**Descrição**</span>: O usuário poderá gravar um som com seu microfone e adicioná-lo ao aplicativo

<span style="color: rgb(218,112,214);">**Regras**</span>:
    - A gravação não deve ter mais do que a duração de 16 ticks



### <span style="color: rgb(218,112,214);">Reproduzir som</span>

<span style="color: rgb(218,112,214);">**Descrição**</span>: O aplicativo deve ser capaz de reproduzir os sons carregados nele, seja por arquivo de áudio ou gravação

<span style="color: rgb(218,112,214);">**Regras**</span>:
    - O aplicativo deve ter um limite máximo de polifonia de um som para consigo mesmo



### <span style="color: rgb(218,112,214);">Remover som</span>

<span style="color: rgb(218,112,214);">**Descrição**</span>: O usuário deve poder remover um som carregado no aplicativo

<span style="color: rgb(218,112,214);">**Regras**</span>:
    - N/A



### <span style="color: rgb(218,112,214);">Substituir som</span>

<span style="color: rgb(218,112,214);">**Descrição**</span>: O usuário deve poder substituir um som carregado por outro, seja este um arquivo ou uma gravação

Nota: Na prática, é só um atalho para remover um som e adicionar outro, mas mais conveniente

<span style="color: rgb(218,112,214);">**Regras**</span>:
    - N/A
